A website to visualize hazards posed by tropical cyclones (TCs) to petrochemical storage tanks in the Houston region. Part of a 
larger collaboration between multiple universities. (Work in progress; will be updated as new analyses are completed)

URL for this fork: https://sustielz.github.io/nas-clim-frontend/

Hazards for this project will include:
- Storm Surge
- Rainfall 
- Wind
- Storage Tank Fragility
- Spill Extent Scenarios
- Public Health Risk
